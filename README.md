# WinForms-Timer-CSharp
# WinForms Timer App — C#

A clean and simple Timer application built with C# Windows Forms.
Designed for beginners learning Event-Driven Programming.

## Features
- Start / Pause / Stop / Reset
- Live counter display
- Minimal and readable code structure
- Built using Event Handler pattern

## Tech Stack
- C# .NET Framework
- Windows Forms (WinForms)
- Visual Studio

## What I Learned
- Event-Driven Programming
- How to use `System.Windows.Forms.Timer`
- Event Handlers (`object sender, EventArgs e`)
- Managing state with class-level variables

## How to Run
1. Clone the repo
2. Open `Timer_App.sln` in Visual Studio
3. Press `F5` to run

## Project Structure
```
Timer_App/
├── Form1.cs           # Logic & Event Handlers
├── Form1.Designer.cs  # UI Controls
└── Timer_App.sln      # Solution file
```
